pBot

A privately hosted, AI-assisted companion bot of your own making.

pBot is a minimal, dockerized, chat bot assisted by:

  • OpenAI's GPT (OpenAI account required)
  • Mid Journey (optionally)

that optimizes for:

  • minimal technical knowledge
  • configuration & extendibility

it can:

  • be managed via a simple web UI
  • adopt whatever persona, rules, and tasks you write
  • respond with images it generates via mid journey
  • adapt to the servers it's in and tailor its responses to the vibe of the server
  • run anywhere. Run it on a spare raspberry pi.

so you can:

  • get started experimenting with AI assistance
  • brainstorm in private
  • easily build on top of a minimal framework
  • annoy your friends

* As private as sending OpenAI conversation snippets get..
